## Haim Tzabar

1. Clone the repository:
```bash
git clone https://github.com/haimtz/myop-work.git
cd myop-work
```

2. install dependencies
```bash
npm ci
```

3. to run the project you will need to run two terminal process 

server
```bash
npm run server
```

gui
```bash
npm run gui
```

4. on browser open localhost:5173
